06How We Handle Your Data

The part most firms ask first. Short version:

  • No retention. Dodonai does not retain firm data after processing.
  • No training. Firm data is never used to train any model. Anthropic's API does not train on inputs by default.
  • Tenant-resident option. Agents run inside your own Microsoft 365 / Google Workspace tenant. Drafts land in your own SharePoint / Drive. No third-party storage.
  • SOC 2-aligned controls. HIPAA-aware document handling. BAA available where required (e.g., medical-records summarization workflows).
  • Privilege stays with the attorney. Every draft is human-reviewed before it leaves the firm. Agents draft and prepare; the attorney reviews and files.

A written compliance brief is available on request before signature.

07What's Different About Working With Us
