The stuck point

The records-upload problem, solved first

This is the one place the current workflow genuinely jams. It's also the cleanest, highest-value thing to build first.

  • Tatiana shares a personal OneDrive folder; firms get "lazy" about using it
  • Practice Panther's portal caps documents at 25 MB, too small for a real record set
  • Large files get chased by email, downloaded, re-sorted, and hand-loaded into Dodonai
→
With the OS
  • One secure upload link per matter, sent to the firm automatically
  • Large files accepted directly, no 25 MB ceiling, no personal folder sharing
  • Files land filed in the right matter folder and queued into Dodonai, untouched by hand
