The LOD point values aren't written by the AI

Every impairment gets looked up in the 2025 Point Chart, and the total is added up by code. So the score and the 9-point threshold call are exact every time. Each distinct injury lands under its own category with the record citation behind it. If something doesn't map cleanly to the chart, it flags it for you instead of guessing.
