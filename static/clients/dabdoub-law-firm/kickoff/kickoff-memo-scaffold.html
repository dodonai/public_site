§6 Data-Handling SOP

This SOP lives inside the Kickoff memo as the durable operating description of how claim files, PHI, and matter data are handled during the engagement. It is the operational companion to the CSA's confidentiality (CSA §5), DPA (Exhibit A), and BAA-where-PHI-is-processed (CSA §4.3) provisions. The CSA controls on any conflict.

What data Dodonai touches. Claim files (often hundreds of pages of medical records, vocational reports, claim forms, correspondence), medical reviews/IME/FCE/CPET reports, denial letters, policies, client PII, and the matter folders that hold them.

Where it stays. Inside the firm's own systems: its Microsoft 365 tenant (mail, calendar, Teams) and its firm-controlled file storage — SharePoint today, migrating to the firm's own Egnyte tenant (the migration target switched from Azure to Egnyte on 2026-06-02). HubSpot holds pipeline metadata only, not file storage. AI workflows read from the matter folder and write .docx output back into the matter folder's "AI output" subfolder. Dodonai introduces no storage of its own outside these firm-controlled systems, and does no Dodonai-side logging of firm content beyond the firm's own audit trail. Egnyte is the firm's own contracted platform; where the engagement relies on it as a sub-processor of firm content, that is reflected in the DPA. This sovereignty, everything under the firm's control, is a value Eddie holds explicitly.

Processing purpose. Customer Data is processed solely to deliver the scoped workflows (CSA §3.5, DPA §2). It is not used for advertising, profiling, or any purpose outside the Services, and it is never used to train, fine-tune, or improve any product or model offered to any other client (CSA §3.5, DPA §2.b.iii).

LLM API handling. For the drafting step, claim-file content routes through Microsoft Graph to the LLM provider and returns as a .docx in the matter folder. This is pass-through inference: the provider does not train on the firm's data (Anthropic zero-data-retention / enterprise terms; OpenAI API no-training-by-default; Exhibit B). LLM API cost is at-cost, no markup, billed via the firm's own provider key or through Dodonai with a per-workflow usage breakdown available. Projected ~$200–$500/month at the Phase 1A stage.

Access. Limited to Key Personnel (Nick) under the engagement (CSA §1.4). Anyone else handling Customer Data or Confidential Information is bound by obligations no less protective than the CSA (CSA §1.7, §5.4).

Accounting Wall-Off

Fee and disbursement data is walled off by the permission model in HubSpot and in any Dodonai dashboard. Accounting data is not surfaced in Appeals-facing views or in firm-wide dashboards. Maria Calvo's accounting function sits behind that wall.

Human-in-the-Loop

No auto-send. Every AI-drafted document is human-reviewed before it leaves the firm; privilege stays with the attorney. Deadline calendaring is always human-supervised. AI may calendar a date, but a human reads the denial letter and confirms it (the default ERISA appeal deadline is 180 days from receipt of denial, but rare exceptions appear in the letter). Submissions and deadlines are never described as fully automated.

Retention / return-or-destroy. On termination or expiration, or earlier on the firm's written request, Dodonai returns or deletes Customer Data within 30 days, unless retention is required by law, and certifies deletion in writing on request (CSA §4.5, DPA §7). At any exit point the firm keeps everything built and receives a portability package in source form (CSA §3.7).

PHI / BAA. Where the Services involve processing PHI under HIPAA, a BAA is executed as a condition precedent and controls on PHI matters (CSA §4.3).

Each phase is a standalone commitment; on convenience termination the firm owes Fees for completed phases plus the in-flight phase's start payment, and nothing for phases not commenced (CSA §10.3 as applied per the SOW). A Material Schedule Failure remedy (CSA §1.10) lets the firm terminate an in-flight phase for cause if a milestone slips materially for reasons in Dodonai's control and is not cured after notice.
